# TicketNow - Database Access Guide

## 📊 How to Access Booking Data from Database

### **Method 1: API Endpoint (Recommended)**

Your backend provides a **GET endpoint** to retrieve all bookings:

```
GET http://localhost:9090/api/bookings
```

**Response Format:**
```json
[
  {
    "id": 1,
    "name": "John Doe",
    "email": "john@example.com",
    "department": "Engineering",
    "numberOfTickets": 3
  },
  {
    "id": 2,
    "name": "Jane Smith",
    "email": "jane@example.com",
    "department": "Marketing",
    "numberOfTickets": 2
  }
]
```

**How to test:**
1. Backend running: `http://localhost:9090/api/bookings`
2. In browser: Just paste the URL
3. In React: 
```javascript
fetch('http://localhost:9090/api/bookings')
  .then(res => res.json())
  .then(bookings => console.log(bookings))
```

### **Method 3: Create a React Bookings Page**

Add this component to view all bookings:

```javascript
import { useState, useEffect } from 'react';

function BookingsList() {
  const [bookings, setBookings] = useState([]);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    fetch('http://localhost:9090/api/bookings')
      .then(res => res.json())
      .then(data => {
        setBookings(data);
        setLoading(false);
      });
  }, []);

  if (loading) return <p>Loading...</p>;

  return (
    <div>
      <h2>All Bookings</h2>
      <table>
        <thead>
          <tr>
            <th>ID</th>
            <th>Name</th>
            <th>Email</th>
            <th>Department</th>
            <th>Tickets</th>
          </tr>
        </thead>
        <tbody>
          {bookings.map(booking => (
            <tr key={booking.id}>
              <td>{booking.id}</td>
              <td>{booking.name}</td>
              <td>{booking.email}</td>
              <td>{booking.department}</td>
              <td>{booking.numberOfTickets}</td>
            </tr>
          ))}
        </tbody>
      </table>
    </div>
  );
}

export default BookingsList;
```

### **Database Schema**

**BOOKING Table:**
```sql
CREATE TABLE BOOKING (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  name VARCHAR(255),
  email VARCHAR(255),
  department VARCHAR(255),
  number_of_tickets INTEGER NOT NULL
);
```

**Columns:**
- `id`: Auto-generated primary key
- `name`: Customer name
- `email`: Customer email
- `department`: Customer department
- `number_of_tickets`: Number of tickets booked

### **Available Endpoints**

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/events` | Get event details |
| POST | `/api/book` | Create a new booking |
| GET | `/api/bookings` | Get all bookings |

---

### **Booking Data Flow**

```
React Form 
    ↓
POST /api/book (Booking data)
    ↓
Backend validates & saves to Database
    ↓
Returns success response
    ↓
GET /api/bookings (Retrieve all bookings)
    ↓
Display in UI
```
